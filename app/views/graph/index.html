Instructions:

  • Click in the open space to add a node, drag from one node to another to add an edge
  • Ctrl-drag a node to move the graph layout
  • Click a node or an edge to select it
  • When a node is selected: Delete removes the node
  • When an edge is selected: Delete removes the edge
  • Each arc is automatically assigned a random number between 0 and 9 as a maximum flow value
  • To assign an arc a new max flow value, select it, type the flow (0-9), then press enter
